# Dessert shop

A dessert-shop built using blazzing-app API + Sanity CMS.
# Features:
+ built in dashboard to manage orders in real-time



# Tech stack
- Nextjs
- Sanity CMS
- Cloudflare workers
- Cloudflare pages
- Hono js
- TailwindCSS (shadcn + vaul)
- Zustand (state-management)
- Blazzing app API
